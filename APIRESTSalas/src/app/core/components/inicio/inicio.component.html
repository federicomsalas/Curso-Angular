Entregable APIREst

Se actualiza aplicación para responder al entregable de APIRest con las siguientes funcionalidades:

- Login de usuario administrador o usuario común

- Utilización de APIs en MockApi para gestionar alumnos, cursos e inscripciones

- LazyLoading

- Guards para la navegación: solo se permite navegar a usuarios autenticados y el menú de cursos solo está disponible para usuarios Administrador.
